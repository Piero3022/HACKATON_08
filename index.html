PREGUNTA 01 y 02 :JavaScript fue creado en 1995 por Brendan Eich mientras
trabajaba en Netscape Communications Corporation. El objetivo era crear un
lenguaje de programación para el lado del cliente que pudiera agregar interactividad a
las páginas web. Eich se basó en el lenguaje de programación Scheme y le agregó elementos de otros
lenguajes como C y Java. El resultado fue un lenguaje interpretado, orientado a objetos y diseñado para la web.
JavaScript se convirtió rápidamente en uno de los lenguajes de programación más populares en el mundo debido a
su capacidad para agregar interactividad y dinamismo a las páginas web. Hoy en día, JavaScript se utiliza en diferentes
ámbitos como el desarrollo web, aplicaciones de escritorio, móviles y creación de software para servidores.

PREGUNTA 03: JavaScript recibió ese nombre porque inicialmente se llamaba Mocha, luego se llamó
LiveScript y finalmente se cambió a JavaScript para aprovechar la popularidad del
lenguaje Java en ese momento. A pesar de su nombre, JavaScript es un lenguaje de programación independiente de Java.
